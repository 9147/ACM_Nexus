ACM Nexus
Unite Through Innovation

 

Event Details:

Event Name: ACM Nexus: Unite Through Innovation

Date and Time: January 5th, 2024, from 1:00 PM to January 6th, 2024, 5:00 PM.

Team Size: 4 members per team

 

Overview:

Theme: Innovation across multiple domains

Objective: Encourage creative solutions to diverse problem statements within the given themes.

 

General Rules:

  • Team Formation:
    1. Teams must consist of exactly 4 members.
  • Eligibility:
    1. Open to all participants interested in problem-solving and innovation.
    2. Participants must abide by the event's Code of Conduct.
  • Project Development:
    1. Projects should align with at least one of the provided problem statements.
    2. Use of open-source libraries and APIs is permitted.

 

Submission:

  • Each team must submit their project by the specified deadline.
  • Submissions should include a presentation (slides or video) and a brief documentation describing the project.

 

Judging Criteria:

Projects will be evaluated based on innovation, technical complexity, relevance to the problem statement, and presentation quality.

 

 

Code of Conduct:

  • Respectful and ethical behaviour is expected from all participants.
  • Plagiarism or using someone else's work without attribution is strictly prohibited.

 

Event Timeline:

  • January 5th, 2024:
    1. 1:00 PM: Event starts.
    2. 3:00 PM: Round 1 of evaluation
    3. 8:00 PM: Round 2 of evaluation
  • January 6th, 2024:
    1. 11:00 AM: Round 3 of evaluation
    2. 1:00 PM: Event ends, project submissions close.
    3. 1:30 PM - 3:30 PM: Project presentations and judging.
    4. 4:00 PM: Winners announced, closing ceremony.

 

Prizes:

Prizes will be awarded to the top-performing teams based on the judging criteria

 

Disclaimer:

ACM KLS GIT reserves the right to modify the event rules and guidelines if deemed necessary. Any changes will be communicated to the participants promptly
